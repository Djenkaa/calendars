<h2>Install</h2>

<p>Run the following commands:</p>

1. composer Install
2. npm install
3. cp .env.example .env
4. configure the env file
5. php artisan key:generate 
6. php artisan config:cache
7. php artisan migrate --seed
8. php artisan serve
